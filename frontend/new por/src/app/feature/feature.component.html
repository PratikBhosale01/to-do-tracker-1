TASK-TRACKER

Features

  • User can add new tasks.
  • Each task will have a priority level (High/Medium/Low) and a due date.
  • Existing tasks can be updated, deleted, marked as 'completed' or moved to archive by the user.
  • User has a search option to search tasks according to the title of the task.
  • The application displays tasks with 'Title', 'Due Date', and 'Due Date Status' by clicking on the Notify button. The due date statuses are 'overdue', 'not due yet', and 'due soon'.
